13. GA4 Custom Dimensions Copy-Paste Table

Use event scope for all of these.

| Display name | Parameter name | Scope | Description |
| --- | --- | --- | --- |
| Locale | locale | Event | Active UI locale used on the page when the event fired. |
| Browser Language | browser_language | Event | Normalized browser language preference, for example `en` or `es`. |
| Browser Language Raw | browser_language_raw | Event | Raw browser language value, for example `es-419` or `pt-BR`. |
| Browser Language Supported | browser_language_supported | Event | Whether the browser language matched a supported site locale. |
| Defaulted Locale Reason | defaulted_locale_reason | Event | Why the site fell back to the default locale. |
| CTA ID | cta_id | Event | Stable identifier for the clicked CTA. |
| CTA Variant | cta_variant | Event | Variant or style bucket for the CTA. |
| CTA Position | cta_position | Event | Placement bucket of the CTA on the page. |
| Section ID | section_id | Event | Stable page section identifier tied to the interaction. |
| Content ID | content_id | Event | Stable content block identifier tied to the interaction. |
| Interaction Type | interaction_type | Event | Type of content interaction, such as open, close, expand, or play. |
| Funnel Step | step | Event | Named funnel step for reservation and conversion flow analysis. |
| Referrer Bucket | referrer_bucket | Event | Normalized source bucket for inbound traffic analysis. |
| Device Bucket | device_bucket | Event | Normalized device class or bucket. |
| Age Range | age_range | Event | Self-reported attendee age-range bucket. |
| Region Bucket | region_bucket | Event | Self-reported attendee region bucket. |
| Attendee Type | attendee_type | Event | Whether the attendee is new or returning. |
| Country | country | Event | Self-reported attendee country code where available. |

16. Best Native GA4 Reports For Marketing

- Acquisition overview
  - dimensions: `sessionSourceMedium`, `sessionDefaultChannelGroup`
  - metrics: `sessions`, `totalUsers`
- Landing page performance
  - dimensions: `landingPage`, `landingPagePlusQueryString`
  - metrics: `sessions`, `totalUsers`
- Geography
  - dimensions: `country`, `city`, `region`
  - metrics: `sessions`, `totalUsers`
- Language and localization
  - dimensions: `language`
  - metrics: `sessions`, `totalUsers`
- Time of day
  - dimensions: `hour`, `dayOfWeekName`
  - metrics: `sessions`, `eventCount`
- Event mix
  - dimensions: `eventName`
  - metrics: `eventCount`
